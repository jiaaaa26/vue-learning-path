# Ajax 学习项目

这个项目包含了学习 Ajax 过程中的三个练习示例，展示了不同方式实现 Ajax 请求的方法。

## 项目结构

## 练习内容

### 1. 基础练习 (Practice1.html)
- 使用 XMLHttpRequest 发送 GET 请求
- 从 JSONPlaceholder API 获取帖子数据
- 在页面上显示帖子内容

### 2. 进阶练习 (Practice2.html)
- 使用 Fetch API 实现搜索功能
- 实现搜索框自动补全
- 包含防抖处理
- 展示搜索结果

### 3. 挑战练习 (Practice3.html)
- 使用 Axios 实现完整的评论区功能
- 支持评论提交
- 实时显示评论列表
- 包含表单验证和错误处理

## 技术特点

- 使用三种不同的 Ajax 实现方式：
  - XMLHttpRequest
  - Fetch API
  - Axios
- 包含错误处理机制
- 实现防抖优化
- 动态 DOM 操作
- 异步编程实践

## 如何运行

1. 克隆项目到本地
2. 使用现代浏览器打开任意 HTML 文件
3. 确保有网络连接（需要访问 JSONPlaceholder API）

## 学习要点

1. Ajax 的基本概念和使用
2. 不同 Ajax 实现方式的对比
3. 异步编程的处理方法
4. 前端性能优化技巧
5. 错误处理最佳实践

## 注意事项

- 项目使用 JSONPlaceholder 作为模拟 API
- 需要现代浏览器支持
- 部分功能需要网络连接
